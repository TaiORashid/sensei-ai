# Sensei Frontend Remade

A modern Next.js application for transforming lecture slides.

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser.

## Features

- Interactive image slider with cursor tracking
- PDF upload functionality
- Modern, clean design with green theme (#034748)
